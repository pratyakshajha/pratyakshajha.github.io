Face Generation

Fourth project on Generative Adversarial Networks(GAN) project in the Deep Learning Nanodegree.

The aim is to use generative adversarial networks to generate new images of faces. A subset of Celeba dataset is used to train the networks.

  • Import data
  • Define generators and discriminator models
  • Detect Dogs
  • Train the networks

The project is implemented using Pytorch in Google Colab. Accompanying Github link is here.
