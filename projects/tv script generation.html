TV Script Generation

Third project on Recurrent Neural Networks (RNN) project in the Deep Learning Nanodegree.

The aim is to generate Seinfeld TV scripts using RNNs using Seinfeld dataset of scripts from 9 seasons. The Neural Network built generates a new, "fake" TV script. Various steps involved are listed below.

  • Import data
  • Preprocess data
  • Build a neural network
  • Train the network
  • Generate TV script

The project is implemented using Pytorch in Google Colab. Accompanying Github link is here.
